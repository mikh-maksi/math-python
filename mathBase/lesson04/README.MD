# Віднімання
Дія віднімання аналогічна дії додавання: ми так само рухаємось вздовж координатної прямої. Проте в випадку із віднімання ми рухаємося вліво (тоді як при додаванні ми рухаємося вправо).

| 8 | - | 7 | = | 1 |
| 7 | - | 4 | = | 3 |
